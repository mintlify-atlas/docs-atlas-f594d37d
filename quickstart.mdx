## Prepare your data

For each image, place two companion files alongside it:

```txt
image.jpg
image.jpg.camera_model.txt
image.jpg.gravity.txt
```

The **gravity file** contains the gravity vector in camera coordinates (x right, y down, z forward):

```txt
0.022717 0.999539 -0.020155
```

The **camera calibration file** contains the model name and comma-separated parameters:

```txt
OPENCV 2575.939766, 2608.293891, 1599.263118, 1257.125449, 0.141865, -0.465301, 0.000000, 0.000000
```

## Run the pipeline

```bash
# Launch the GUI
src/exe/ppsfm gui

# Or run all stages from the command line:
ppsfm feature_extractor --database_path db.db --image_path images/
ppsfm exhaustive_matcher --database_path db.db
ppsfm mapper --database_path db.db --image_path images/ --output_path results/
```
