## Privacy Preserving Structure-from-Motion

PP-SfM is a modified COLMAP pipeline from the [Computer Vision and Geometry Group](http://www.cvg.ethz.ch) at ETH Zurich. It enables 3D reconstruction while protecting image privacy by replacing standard keypoint positions with **line features** — so a reconstruction server never sees exact pixel locations of people, vehicles, or other sensitive content.

## How it works

Instead of sending keypoint pixel coordinates `(x, y)` to a server, PP-SfM converts each keypoint to a line `l = (a, b, c)^T` satisfying `ax + by + c = 0`. The line passes through the original keypoint, but the exact position remains on the client.

Lines whose orientation is parallel to the gravity direction are flagged as **gravity-aligned** and provide stronger reconstruction constraints. The pipeline uses a 4-view initialization that requires both aligned and unaligned tracks.

<Note>
  Gravity direction and camera calibration files must be provided alongside each image. See the [Quickstart](/getting-started/quickstart) for the required file format.
</Note>
